<h5>运行我们的B程序</h5>

1. 你可以从右侧的 Releases（版本）下载启动器。

2. 下载源代码。
   - 点击 Code -> Download ZIP，下载源代码的 zip 压缩包。

   - 解压压缩包。

   - 确保你有 Python 3.12 作为 Python 解释器。

   - 运行 `pip install -r requirements.txt`。

   - 运行 `python main.py`。
